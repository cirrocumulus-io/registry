# Cirrocumulus Registry
Provide a REST API to share image used by Cirrocumulus.

## How to build
```shell
./gradlew assemble
```

## How to execute
- With Gradle:
```shell
./gradlew run
```

- With Java
```shell
java -jar api/build/libs/repository-api-0.1.0-SNAPSHOT.jar
```

## How to contribute
Please open an issue to propose your feature!
